# CVA6 BAREMETAL BSP

## Get started

CVA6 BAREMETAL BSP project purpose is to provides baremetal support for the CVA6 COREV APU on GENESYS2. More info on this target can be found in the [CVA6 repository](https://github.com/openhwgroup/cva6).
